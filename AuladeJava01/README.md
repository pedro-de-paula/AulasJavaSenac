# ☕ Primeiros Passos com Java

Este projeto contém uma pequena coleção de arquivos `.java` criados com o propósito de **ensinar** e **praticar** os fundamentos da linguagem Java. Cada arquivo representa um conceito essencial para quem está dando os primeiros passos na programação.

---

## 📂 O que tem aqui?

### 🧱 `Aula01.java`
O famoso “Hello, World!”... e além!  
Aqui você encontra a estrutura básica de um programa em Java:
- Método `main`
- Impressão com `System.out.println`
- Comentários no código
- Primeiro contato com a sintaxe da linguagem

### 🧠 `Variáveis.java`
Tudo começa com dados. Este arquivo mostra como criar e usar variáveis de diferentes tipos:
- `int`, `double`, `boolean`, `String`
- Declaração e inicialização
- Dicas sobre boas práticas de nomes

### ➕ `OperadoresBasicos.java`
Matemática básica com Java:
- Operadores: soma, subtração, multiplicação, divisão e módulo
- Operações simples entre variáveis
- Ordem de precedência (sim, parênteses importam!)

📚 Para que serve isso?

Este repositório é ideal para:

    Estudantes iniciantes em Java

    Quem precisa revisar a base da linguagem

    Professores que querem exemplos simples para sala de aula

    Gente curiosa querendo entender “como esse tal de Java funciona”

🛠️ Requisitos

    Java 8 ou superior (qualquer versão moderna funciona)

    Terminal ou Prompt de Comando

    Um editor legal (VS Code, IntelliJ, Eclipse, ou o clássico bloco de notas)
