# 🍝 Receita de Macarronada em Java

> Um programa em Java que simula o preparo de uma macarronada passo a passo.  
> Desenvolvido por: Matheus Alves, Pedro Augusto, Pedro de Paula, Rafaela Leite e Sabrina de Sá.

---

## 🧾 Sobre o Projeto

Este repositório contém um programa que **ensina como fazer uma macarronada**, utilizando comandos Java como se você estivesse lendo um script de culinária automatizado. Ele usa strings, variáveis, estruturas condicionais e o clássico `Scanner` para interatividade no terminal.

Você escolhe se quer queijo ralado ou não. 🍝🧀

---

## 📜 O que você vai aprender

- Declaração e uso de variáveis `String`
- Estrutura sequencial de instruções
- Entrada de dados com `Scanner`
- Uso de `if/else` com `equalsIgnoreCase()`
- Organização lógica em Java

---

## 📂 Arquivo principal

### `Receita.java`

Este arquivo contém:

✅ Uma sequência de **22 passos** para fazer uma macarronada  
✅ Interatividade com o usuário (`Quer queijo ralado?`)  
✅ Variáveis bem nomeadas para ingredientes e utensílios  
✅ Saída divertida no console

1. Certifique-se de ter o Java JDK instalado.
